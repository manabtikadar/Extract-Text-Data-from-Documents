Welcome to My Project: "Intelligently Extract Text & Data from Documents with OCR NER"!

In this project, I have developed a customized Named Entity Recognizer (NER). The primary goal is to extract entities from scanned documents such as invoices, business cards, shipping bills, and bills of lading. For data privacy reasons, the demonstration focuses on business cards. However, the framework can be extended to process various financial documents.

Below is the development plan and approach followed in the project. The project integrates two core technologies from data science:

  1. Computer Vision
  2. Natural Language Processing (NLP)

In the Computer Vision module, the system scans the document, identifies text locations, and extracts text from images. The NLP module then extracts entities, cleans the text, and parses the extracted information.

Python Libraries Used in the Computer Vision Module:

  • OpenCV
  • NumPy
  • Pytesseract

Python Libraries Used in the NLP Module:

  • SpaCy
  • Pandas
  • Regular Expressions
  • String Manipulations

To ensure clarity and structure, the project was divided into several stages:

Stage 1: Setting Up the Project
  • Installing Python
  • Installing Dependencies
Stage 2: Data Preparation and Text Extraction
  • Gathering Images
  • Overview of Pytesseract
  • Extracting Text from Images
  • Cleaning and Preparing Text
Stage 3: Data Labeling for NER Using BIO Tagging
  • Manual Labeling with the BIO Technique:
    • B - Beginning
    • I - Inside
    • O - Outside
Stage 4: Data Preprocessing for Machine Learning
  • Preparing Training Data for SpaCy
  • Converting Data into SpaCy Format
Stage 5: Training the Named Entity Model
  • Configuring the NER Model
  • Training the Model
Stage 6: Prediction, Entity Extraction, and Text Parsing
  • Loading the Model
  • Rendering and Serving with Displacy
  • Drawing Bounding Boxes on Images
  • Parsing Entities from Text

Finally, I integrated all components to create a comprehensive document scanner application.
